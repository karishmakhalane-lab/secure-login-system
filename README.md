# secure-login-system
A Python-based Secure Login System that implements user authentication, password hashing, account lockout after failed login attempts, input validation, and activity logging to demonstrate cybersecurity best practices.
# Project purpose
The purpose of this project is to develop a secure login system that authenticates users before granting access to an application. The system allows users to register with a username and password and verifies their credentials during login. Passwords are stored in a secure hashed format instead of plain text to improve security. The project demonstrates the fundamentals of user authentication, password protection, and secure access control while providing a simple and user-friendly interface.
